# Xray-tumor-segmentation
 X-ray Tumor Segmentation System 🩻

 📌 Project Overview

This module is part of a multi-modal medical imaging system. It focuses on tumor segmentation using X-ray images.

 🧠 Model Used

 U-Net (Deep Learning segmentation model)

 ⚙️ Features

* X-ray image preprocessing
* Tumor segmentation using deep learning
* Visualization of predicted masks

 📂 Project Structure

```
model/        → U-Net model  
utils/        → Dataset loader  
train.py      → Training script  
predict.py    → Prediction script  
```

 ▶️ How to Run

 Install dependencies:

```
pip install -r requirements.txt
```

 Train:

```
python train.py
```

 Predict:

```
python predict.py
```
